TermUtil
========

Terminal utilities for C-language.

This library enables to change foreground or background color, to move cursor
and others on terminal.

This library supports both MSVC and gcc.


## Build

#### gcc

```sh
$ make
```

#### MSVC

```sh
> nmake /f msvc.mk
```
